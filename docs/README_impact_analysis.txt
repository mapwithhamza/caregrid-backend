CareGrid India — Impact / Desert Analysis Package

Use these files for dashboard insights, README, pitch deck, and backend impact endpoints.

1. caregrid_desert_state_risk_index.csv
State-level trust desert index.
Use for state map layer, risk ranking, and state detail panel.

2. caregrid_desert_priority_states.csv
Action-oriented priority state table.
Use for NGO/planner action list.

3. caregrid_desert_calibrated_priority_ranking.csv
Sharper national priority ranking.
Use this for judge-facing conclusions and final pitch.
Tier 1 national immediate priority states:
Maharashtra, Uttar Pradesh, Gujarat, Bihar, Tamil Nadu.

4. caregrid_desert_trust_gap_summary.csv
One-row national summary.
Use for KPI cards, headline insight, README, and pitch.

5. caregrid_desert_facility_type_gap.csv
Facility-type reliability analysis.
Use for chart showing reliability gaps by clinic/hospital/dentist/doctor/pharmacy.

Important interpretation:
This is a preliminary TRUST desert analysis, not final specialty desert analysis.
Final specialty desert analysis should be added after Plan A capability extraction is completed.
